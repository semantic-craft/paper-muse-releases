# PaperMuse Releases

Public distribution surface for [PaperMuse](https://github.com/semantic-craft/paper-muse) on macOS.

GitHub Releases in this repository contain:

- Developer ID signed, Apple-notarized, stapled PaperMuse application archives.
- Versioned Apple Silicon Python runtime archives used by the application.
- Release notes and SHA-256 checksums for published assets.

Source code, issues, and pull requests remain in the [paper-muse source repository](https://github.com/semantic-craft/paper-muse). No credentials or user data belong in this repository.
